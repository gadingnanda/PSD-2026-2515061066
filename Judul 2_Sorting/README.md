Judul : Pengurutan harga buku <br>

Deskripsi : <br>
Program ini menggunakan Bubble Sort untuk mengolah data harga buku dengan cara mengurutkannya dari nilai terkecil ke terbesar. Pengguna diminta untuk memasukkan jumlah buku, kemudian memasukkan harga masing-masing buku, kemudian data yang telah dimasukkan akan ditampilkan sebelum dan sesudah proses pengurutan. <br>
<br>
<br>

Fungsi ini digunakan untuk menukar posisi dua elemen dalam sebuah list. Fungsi ini menerima tiga parameter yaitu arr sebagai list data, menggunakan i dan j sebagai indeks dari elemen yang ingin ditukar. Proses penukaran dilakukan dengan menggunakan variabel sementara yaitu temp untuk menyimpan salah satu nilai agar tidak hilang saat proses pertukaran berlangsung. Nilai pada indeks i dipindahkan ke indeks j, dan sebaliknya.<br>

Fungsi ini berfungsi untuk mengurutkan data dalam list menggunakan algoritma Bubble Sort secara ascending (dari kecil ke besar). Fungsi ini bekerja dengan cara membandingkan dua elemen yang berdekatan dalam list, kemudian menukar posisinya jika urutannya salah.<br>

<br>

Fungsi ini membuat program dimana pengguna diminta untuk memasukkan jumlah data dan melakukan validasi menggunakan try-except untuk memastikan input berupa angka. Setelah itu, program meminta pengguna memasukkan harga buku satu per satu sesuai dengan jumlah data sebelumnya dan menyimpannya ke dalam list, dengan validasi input agar tidak terjadi kesalahan. Selanjutnya, program menampilkan daftar harga sebelum diurutkan, kemudian memanggil fungsi bubble_sort untuk mengurutkan data. Setelah proses sorting selesai, program menampilkan kembali data yang sudah terurut.<br>
