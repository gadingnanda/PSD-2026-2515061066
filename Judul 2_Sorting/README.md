Judul : Pengurutan harga buku <br>

Deskripsi : <br>
Program ini menggunakan Bubble Sort untuk mengolah data harga buku dengan cara mengurutkannya dari nilai terkecil ke terbesar. Pengguna diminta untuk memasukkan jumlah buku, kemudian memasukkan harga masing-masing buku, kemudian data yang telah dimasukkan akan ditampilkan sebelum dan sesudah proses pengurutan.
